# Sound Files

Place the following sound files in this directory:

## Required Files

| File | Description | Recommended |
|------|-------------|-------------|
| `fart1.mp3` | Fart sound variation 1 | Short, comedic toot |
| `fart2.mp3` | Fart sound variation 2 | Different pitch/length |
| `fart3.mp3` | Fart sound variation 3 | Another variation |
| `coin.mp3` | Coin collect sound | Short, bright ding |
| `purchase.mp3` | Purchase success | Cash register or ka-ching |
| `levelup.mp3` | Level up fanfare | Short celebratory jingle |
| `prestige.mp3` | Prestige achievement | Epic/triumphant sound |
| `click.mp3` | UI click sound | Soft click/tap |
| `music.mp3` | Background music | Loopable, cheerful tune |

## Free Sound Resources

- [Freesound.org](https://freesound.org) - Search for "fart", "coin", etc.
- [Zapsplat](https://www.zapsplat.com) - Free sound effects
- [Pixabay](https://pixabay.com/sound-effects/) - Royalty-free sounds

## Notes

- All sounds should be MP3 format for best browser compatibility
- Keep file sizes small (under 100KB for SFX, under 2MB for music)
- Music should be seamlessly loopable
- Consider using short, punchy sounds for better game feel
